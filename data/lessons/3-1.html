What is a Quadratic Function?

A quadratic function is a function that can be written in standard form:

f(x) = ax² + bx + c,  where a ≠ 0

Its graph is a U-shaped curve called a parabola.

parabolaThe U-shaped graph of a quadratic function.
vertexThe highest or lowest point of a parabola — where the graph turns around.
axis of symmetryThe vertical line x = −b/(2a) that cuts the parabola into two mirror-image halves. It always passes through the vertex.
y-interceptWhere the graph crosses the y-axis. For f(x) = ax² + bx + c it is always the point (0, c).
maximum / minimumThe y-value of the vertex. It is a minimum if the parabola opens up (a > 0), a maximum if it opens down (a < 0).
average rate of changeHow much f(x) changes per unit of x over an interval [a, b]: (f(b) − f(a)) ÷ (b − a).

Anatomy of a Parabola

Every feature you need to identify, on one picture:

a > 0 → opens up, vertex is a minimum, range: y ≥ k
a < 0 → opens down, vertex is a maximum, range: y ≤ k
Watch out! The domain of every quadratic function is all real numbers — only the range is restricted by the vertex.

How to Graph a Quadratic (5 Steps)

  1. Find the axis of symmetry: x = −b/(2a).
  2. Find the vertex: substitute that x-value into f(x) to get the y-coordinate.
  3. Plot the y-intercept (0, c) — and its mirror point across the axis of symmetry.
  4. Make a table of values with points on both sides of the vertex.
  5. Draw a smooth U-shaped curve through the points. State the domain and range.

Average Rate of Change

For nonlinear functions the rate of change is not constant, so we measure it over an interval:

average rate of change = ( f(b) − f(a) ) / ( b − a )  over [a, b]

Geometrically, this is the slope of the line connecting the two points (a, f(a)) and (b, f(b)) on the graph — try it live in the Rate of Change lab.

  1. Axis of symmetry: x = −b/(2a) = −6/(2·1) = −3
  2. Vertex: f(−3) = (−3)² + 6(−3) + 8 = 9 − 18 + 8 = −1 → vertex (−3, −1)
  3. y-intercept: (0, 8); its mirror across x = −3 is (−6, 8)
  4. Table of values: f(−5)=3, f(−4)=0, f(−3)=−1, f(−2)=0, f(−1)=3
  5. Draw the curve through the points — it opens up since a = 1 > 0.
Check: the points (−4, 0) and (−2, 0) are mirror images across x = −3 — the symmetry confirms the graph.
Domain: all real numbers  ·  Range: y ≥ −1

  1. Look at a: a = −3 < 0, so the parabola opens down → the vertex is a maximum.
  2. x-coordinate of vertex: x = −b/(2a) = −(−9)/(2·−3) = 9/(−6) = −1.5
  3. Maximum value: f(−1.5) = −3(2.25) − 9(−1.5) + 2 = −6.75 + 13.5 + 2 = 8.75
Watch out! Two sign traps: −b means take the opposite of b, and 2a keeps a's negative sign. −(−9) = +9 and 2(−3) = −6.
Maximum value: 8.75 at x = −1.5  ·  Range: y ≤ 8.75

g(x) from the table: g(−10)=170, g(−5)=70, g(0)=10, g(5)=−10, g(10)=10

  1. Minimum of f: axis x = −(−10)/(2·1) = 5; f(5) = 25 − 50 + 5 = −20. Minimum of f is −20.
  2. Minimum of g: the table is symmetric around x = 5 (g(0) = g(10) = 10), so the vertex is at (5, −10). Minimum of g is −10.
  3. Compare: −20 < −10.
f(x) has the lesser minimum: −20 < −10
  1. Set up: ( f(4) − f(−4) ) / ( 4 − (−4) )

Table: f(−3)=48, f(−2)=21, f(−1)=0, f(0)=−15, f(1)=−24, f(2)=−27, f(3)=−24

  1. Read the endpoints only: f(3) = −24 and f(−3) = 48. The middle rows are not needed!
  2. Substitute: ( f(3) − f(−3) ) / ( 3 − (−3) ) = ( −24 − 48 ) / 6
  3. Simplify: −72 / 6 = −12
Average rate of change = −12

A graph of spending shows the points (0, 15.72) and (10, 20.99).

  1. Estimate from the graph: the y-values rise about 5.5 while x changes by 10 → roughly 5.5/10 = 0.55.
  2. Check algebraically: ( f(10) − f(0) ) / ( 10 − 0 ) = ( 20.99 − 15.72 ) / 10 = 5.27/10
  3. Simplify: = 0.527 — close to the estimate, so the estimate was reasonable.
Check: the algebraic answer 0.527 ≈ the graphical estimate 0.55.
