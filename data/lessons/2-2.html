Core Concepts

📏 What is Absolute Value?

The absolute value of a number is its distance from zero on the number line. Distance is always non-negative.

Formal Definition For any real numbers $a$ and $b$, if $|a| = b$ and $b \ge 0$, then $a = b$ or $a = -b$.

This means every absolute value equation with a positive right side has two possible cases to consider.

⚠️ Special Solution Sets
  • Two solutions — the standard case when both cases check out.
  • One solution (extraneous) — one case checks out, the other fails when substituted back. Discard the failing answer; it is an extraneous solution.
  • No solution (empty set) — absolute value can never equal a negative number. The solution set is $\{\}$ or $\varnothing$.

Solving Absolute Value Equations

🔢 The 4-Step Method
1
Isolate

Get the absolute value expression alone on one side of the equation.

2
Write Two Cases

Set the inside expression equal to + and − the right side.

3
Solve Each Case

Apply algebraic properties to solve both equations.

4
Check Solutions

Substitute each answer into the original equation to verify. Discard any extraneous solutions.

Example 1
Two Solutions — Solve $2|5x + 1| - 9 = 4x + 17$
▼
1
Isolate the absolute value
$2|5x + 1| - 9 = 4x + 17$
$2|5x + 1| = 4x + 26$ ← add 9 to both sides
$|5x + 1| = 2x + 13$ ← divide both sides by 2
2
Write two cases
Case 1: positive
$5x + 1 = 2x + 13$
$3x = 12$
$x = 4$
Case 2: negative
$5x + 1 = -(2x + 13)$
$5x + 1 = -2x - 13$
$7x = -14$
$x = -2$
⚠️ Watch Out — Distribute the Negative!

In Case 2, the negative sign applies to the entire right side. Use the Distributive Property: $-(2x + 13) = -2x - 13$.

3
Check both solutions
✓ Verify $x = 4$
$2|5(4) + 1| - 9 = 4(4) + 17$
$2|21| - 9 = 33 \Rightarrow 33 = 33$ ✓
✓ Verify $x = -2$
$2|5(-2) + 1| - 9 = 4(-2) + 17$
$2|-9| - 9 = 9 \Rightarrow 9 = 9$ ✓
Solution set: $\{4, -2\}$
Both values are plotted as filled dots on the number line.
Example 2
Extraneous Solution — Solve $2|x + 1| - x = 3x - 4$
▼
1
Isolate
$2|x + 1| - x = 3x - 4$
$2|x + 1| = 4x - 4$ ← add x to both sides
$|x + 1| = 2x - 2$ ← divide by 2
2
Write two cases and solve
Case 1
$x + 1 = 2x - 2$
$3 = x$
$x = 3$
Case 2
$x + 1 = -(2x - 2)$
$x + 1 = -2x + 2$
$3x = 1$
$x = \frac{1}{3}$
3
Check — one solution fails!
✓ $x = 3$ checks out
$2|3 + 1| - 3 = 3(3) - 4 \Rightarrow 5 = 5$ ✓
✗ $x = \frac{1}{3}$ is extraneous
$2\left|\frac{1}{3} + 1\right| - \frac{1}{3} = 3\left(\frac{1}{3}\right) - 4 \Rightarrow \frac{7}{3} \ne -3$ ✗
Solution set: $\{3\}$ — $x = \frac{1}{3}$ is discarded as an extraneous solution.
Example 3
Empty Set — Solve $|4x - 7| + 10 = 2$
▼
1
Isolate
$|4x - 7| + 10 = 2$
$|4x - 7| = -8$ ← subtract 10
2
Recognize the impossible case

Absolute value is always $\ge 0$. It can never equal a negative number like $-8$.

No further work is needed — there is no solution.

Solution set: $\varnothing$ (empty set) — no value of $x$ satisfies this equation.

Solving Absolute Value Inequalities

🔀 The Two Rules: < vs >

Absolute value inequalities split into compound inequalities based on the inequality symbol:

FormRewrite AsJoined byGraph
$|ax + b| < c$$-c < ax + b < c$ANDSegment between two values
$|ax + b| \le c$$-c \le ax + b \le c$ANDSegment (endpoints included)
$|ax + b| > c$$ax + b > c$ OR $ax + b < -c$ORTwo rays pointing outward
$|ax + b| \ge c$$ax + b \ge c$ OR $ax + b \le -c$ORTwo rays (endpoints included)
Memory Trick "Less than" → AND (between) — the solution is sandwiched between two values.
"Greater than" → OR (outside) — the solution fans out in two directions.

Example 4
Less-Than Inequality — Solve $|4x - 8| - 5 < 11$
▼
1
Isolate
$|4x - 8| - 5 < 11$
$|4x - 8| < 16$ ← add 5
2
Rewrite as AND compound inequality
$-16 < 4x - 8 < 16$
3
Solve both parts
Left side
$4x - 8 > -16$
$4x > -8$
$x > -2$
Right side
$4x - 8 < 16$
$4x < 24$
$x < 6$
Solution: $\{x \mid -2 < x < 6\}$
Open circles at $-2$ and $6$; shade the region between them.
Example 5
Greater-Than Inequality — Solve $\frac{|6x + 3|}{2} + 5 \ge 14$
▼
1
Isolate
$\frac{|6x + 3|}{2} + 5 \ge 14$
$\frac{|6x + 3|}{2} \ge 9$ ← subtract 5
$|6x + 3| \ge 18$ ← multiply by 2
2
Rewrite as OR compound inequality
$6x + 3 \ge 18$   OR   $6x + 3 \le -18$
3
Solve both parts
Case 1
$6x + 3 \ge 18$
$6x \ge 15$
$x \ge \frac{5}{2}$
Case 2
$6x + 3 \le -18$
$6x \le -21$
$x \le -\frac{7}{2}$
Solution: $\{x \mid x \le -\frac{7}{2} \text{ or } x \ge \frac{5}{2}\}$
Filled dots at $-\frac{7}{2}$ and $\frac{5}{2}$; shade outward in both directions.
🔑 Critical Reminder: Isolate First, Then Split

Always isolate the absolute value expression before rewriting as a compound inequality. If you split too early, you'll get incorrect cases. Also: when multiplying or dividing by a negative number while solving the compound inequality, reverse the inequality symbol.
