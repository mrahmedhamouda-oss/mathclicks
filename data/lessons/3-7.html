Key Concepts

🔑 Quadratic Inequalities in Two Variables

A quadratic inequality is any inequality of the form:

y > ax² + bx + c
y ≥ ax² + bx + c
y < ax² + bx + c
y ≤ ax² + bx + c
ℹ Solid vs Dashed Parabola:
• < or > → dashed parabola (boundary not included)
• ≤ or ≥ → solid parabola (boundary included)

🔑 Steps to Graph a Quadratic Inequality

  1. Graph the related function y = ax² + bx + c (solid if ≤/≥, dashed if </>)
  2. Test a point NOT on the parabola — (0, 0) is easiest if it's not on the curve
  3. Shade the region that contains the point (if test is TRUE); shade the other region (if FALSE)

🔑 Solving Quadratic Inequalities (1 Variable) — Summary

Inequalitya > 0 (opens up)a < 0 (opens down)
ax² + bx + c < 0{x | x₁ < x < x₂}{x | x < x₁ or x > x₂}
ax² + bx + c > 0{x | x < x₁ or x > x₂}{x | x₁ < x < x₂}
ax² + bx + c ≤ 0{x | x₁ ≤ x ≤ x₂}{x | x ≤ x₁ or x ≥ x₂}
ax² + bx + c ≥ 0{x | x ≤ x₁ or x ≥ x₂}{x | x₁ ≤ x ≤ x₂}

Where x₁ and x₂ are the zeros (roots) of the related equation.

🔑 Algebraic Method (Number Line)

  1. Write inequality in standard form: ax² + bx + c ≤ 0 (or similar)
  2. Solve the related equation ax² + bx + c = 0 to find the zeros
  3. Plot the zeros on a number line
  4. Test a value from each interval on the number line
  5. Write the solution set including intervals where the inequality is satisfied
