🔑 End Behavior of Monomial Functions

End behavior describes what happens to f(x) as x → −∞ and as x → +∞.

📈 Even degree, positive a

As x → −∞, f(x) → +∞

As x → +∞, f(x) → +∞

Both ends go UP ↑↑

📉 Even degree, negative a

As x → −∞, f(x) → −∞

As x → +∞, f(x) → −∞

Both ends go DOWN ↓↓

↗ Odd degree, positive a

As x → −∞, f(x) → −∞

As x → +∞, f(x) → +∞

Rises right, falls left ↓↑

↙ Odd degree, negative a

As x → −∞, f(x) → +∞

As x → +∞, f(x) → −∞

Falls right, rises left ↑↓

📌 Key facts on zeros:
• Odd-degree monomial functions always have at least one real zero
• Even-degree monomial functions may have zero or more real zeros
• The degree tells the MAXIMUM number of times the graph crosses the x-axis

🔑 Standard Form of a Polynomial

f(x) = aₙxⁿ + aₙ₋₁xⁿ⁻¹ + … + a₂x² + a₁x + a₀

• Degree = n (highest exponent)
• Leading coefficient = aₙ (coefficient of highest-degree term)
• For a polynomial to be valid: all exponents must be non-negative integers
