Key Concepts

🔑 Square Root Property

To solve a quadratic equation in the form x² = n, take the square root of each side.

For any number n ≥ 0: if x² = n, then x = ±√n

Example: x² = 121 → x = ±√121 = ±11
⚠ Watch Out! Always include the ± symbol before the radical. Both the positive and negative roots are solutions.

🔑 Completing the Square

To complete the square for the expression x² + bx:

  1. Find one half of b, the coefficient of x: compute b/2
  2. Square the result from Step 1: compute (b/2)²
  3. Add the result of Step 2 to x² + bx: x² + bx + (b/2)² = (x + b/2)²
x² + bx + (b/2)² = (x + b/2)²

🔑 Vertex Form of a Quadratic Function

Any quadratic in standard form can be rewritten in vertex form by completing the square.

Standard Form

y = ax² + bx + c

Vertex Form

y = a(x - h)² + k
Vertex: (h, k)
  • If a > 0 → parabola opens up, vertex is a minimum
  • If a < 0 → parabola opens down, vertex is a maximum
  • Axis of symmetry: x = h

🔑 Projectile Motion

The height h(t) of an object at time t is modeled by:

h(t) = -½gt² + vt + h₀

g = 9.8 m/s² or 32 ft/s² (gravity)
v = initial velocity
h₀ = initial height

The vertex gives the maximum height and the time it occurs.

🔑 Solving by Completing the Square (when a ≠ 1)

  1. Divide every term by a (the coefficient of x²)
  2. Move the constant to the right side
  3. Add (b/2)² to both sides
  4. Factor the left side as a perfect square
  5. Apply the Square Root Property
  6. Solve for x

Worked Examples

Solve x² − 4x + 4 = 25 using the Square Root Property.
x² − 4x + 4 = 25Original equation
(x − 2)² = 25Factor left side
x − 2 = ±√25Square Root Property
x − 2 = ±5√25 = 5
x = 2 + 5 = 7 OR x = 2 − 5 = −3Add 2 to each side
Solution Set: {−3, 7}
Solve x² + 24x + 144 = 192 using the Square Root Property.
x² + 24x + 144 = 192Original equation
(x + 12)² = 192Factor
x + 12 = ±√192Square Root Property
x + 12 = ±8√3√192 = 8√3
x = −12 ± 8√3Subtract 12
⚠ Note: 192 is NOT a perfect square, so roots are irrational.
Exact: x = −12 − 8√3 ≈ −25.86    or    x = −12 + 8√3 ≈ 1.86
Solve 2x² − 92x + 1058 = −72 using the Square Root Property.
2x² − 92x + 1058 = −72Original equation
x² − 46x + 529 = −36Divide each side by 2
(x − 23)² = −36Factor
x − 23 = ±√(−36)Square Root Property
x − 23 = ±6i√(−36) = 6i
x = 23 ± 6iAdd 23 to each side
⚠ Complex Roots: When the right side is negative after applying the Square Root Property, roots are complex (involve i = √(−1)).
x = 23 + 6i   and   x = 23 − 6i
Find c so that x² − 7x + c is a perfect square trinomial. Write as a perfect square.
b = −7Identify b
b/2 = −7/2 = −3.5Step 1: Find half of b
(−3.5)² = 12.25Step 2: Square it
c = 12.25Step 3: That's the value of c

Solve x² + 18x − 4 = 0 by completing the square.
x² + 18x − 4 = 0Original equation
x² + 18x = 4Add 4 to each side
b/2 = 9   →   (b/2)² = 81Complete the square
x² + 18x + 81 = 4 + 81 = 85Add 81 to both sides
(x + 9)² = 85Factor
x + 9 = ±√85Square Root Property
x = −9 ± √85Subtract 9
Solution Set: {−9 − √85, −9 + √85}
Solve 4x² − 12x − 27 = 0 by completing the square.
4x² − 12x − 27 = 0Original equation
x² − 3x − 27/4 = 0Divide each side by 4
x² − 3x = 27/4Add 27/4 to each side
b/2 = −3/2   →   (b/2)² = 9/4Complete the square
x² − 3x + 9/4 = 27/4 + 9/4 = 36/4 = 9Add 9/4 to both sides
(x − 3/2)² = 9Factor
x − 3/2 = ±3Square Root Property
x = 3/2 + 3 = 9/2    OR    x = 3/2 − 3 = −3/2Solve for x
Solution Set: {−3/2, 9/2}
Solve 3x² − 72x + 465 = 0 by completing the square.
3x² − 72x + 465 = 0Original equation
x² − 24x + 155 = 0Divide by 3
x² − 24x = −155Subtract 155
x² − 24x + 144 = −155 + 144 = −11Add (−24/2)² = 144
(x − 12)² = −11Factor
x − 12 = ±√(−11) = ±i√11Square Root Property
x = 12 ± i√11Add 12
x = 12 + i√11   and   x = 12 − i√11
Write y = −x² − 12x − 9 in vertex form.
y = −x² − 12x − 9Original function
y = −(x² + 12x) − 9Factor out −1 from x terms
y = −(x² + 12x + 36) − 9 + 36Add (12/2)² = 36 inside; compensate outside
y = −(x + 6)² + 27Simplify
⚠ Key: When a ≠ 1, after factoring out a, balance the equation: adding (b/2)² inside the parentheses means adding a·(b/2)² outside.
Vertex Form: y = −(x + 6)² + 27    Vertex: (−6, 27) — maximum
Consider y = 3x² − 12x + 5. Find vertex form, axis of symmetry, and classify vertex.
y = 3x² − 12x + 5Original
y = 3(x² − 4x) + 5Factor out 3
y = 3(x² − 4x + 4) + 5 − 12Add (−4/2)² = 4 inside; subtract 3·4 = 12 outside
y = 3(x − 2)² − 7Simplify
Vertex Form: y = 3(x − 2)² − 7
Vertex: (2, −7) — minimum (a = 3 > 0)
Axis of Symmetry: x = 2
